# **App Name**: STX TipJar

## Core Features:

- Wallet Connection: Connect to Stacks wallet (Leather / Xverse) using Stacks.js for transaction signing.
- Recipient Address Input: Enter the Stacks address of the tip recipient.
- STX Amount Input: Enter the amount of STX to tip the recipient.
- On-Chain Tip: Send STX tip on-chain via a Clarity smart contract that validates and records the transaction.
- Total Tips Received Display: Calculate and display the total STX tips received per address.
- Tip Count Display: Display the number of tips received per address.
- Recent Tip History: Display recent tip timestamps for an address.

## Style Guidelines:

- Primary color: Bitcoin orange (#F7931A) for a strong, energetic feel.
- Background color: Light gray (#EEEEEE) for a clean, minimal aesthetic.
- Accent color: Darker orange (#D35400) for interactive elements and highlights.
- Body and headline font: 'Inter' sans-serif font known for readability.
- Use simple, outlined icons for key actions and information displays.
- Clean and responsive layout, optimized for mobile and desktop viewing.
- Subtle animations to indicate loading and transaction confirmation.
